## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm

### Installation
```bash
npm install
```

### Development
Start the local development server:
```bash
npm run dev
```
Visit [http://localhost:8080](http://localhost:8080) in your browser.

### Build
To build for production:
```bash
npm run build
```

### Lint
To lint the codebase:
```bash
npm run lint
```
